Column description
===============

subj —> subject id
tr_id —> trial id (formatted as blockid.trialid_inblock)
tr_type —> F, H or L (Face, House or Leaf)
x1… x5 —> x position of fixation number 1… 5
y1… y5 —> y position of fixation number 1… 5
dur1… dur5 —> duration of fixation number 1… 5
gap1… gap5 —> duration between current fixation and next fixation (if large, it likely means there was a blink or missing data between fixations; trials with large gaps are not selected while generating gaze paths)
